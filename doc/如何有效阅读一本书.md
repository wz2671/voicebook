# 超实用笔记读书法 · 操作流程文档


## 一、核心理念（读前必读）

本流程提炼自奥野宣之《如何有效阅读一本书：超实用笔记读书法》。该方法的核心思想是：**用一本笔记管理读书生活的全部环节**，贯穿选书、购书、读书、记录、活用五个阶段，最终目标是“把书中的知识化为己有，在日常中运用自如”。

三个底层原则贯穿始终：

1. **信息一元化**：所有与读书有关的信息（想读的书、读过的书、摘抄、感想、书评剪报等）都记在同一本笔记本里，不设分册、不做分类，按时间顺序记录。
2. **以输出倒逼输入**：始终带着“为写笔记而读书”的意识阅读，主动寻找书中打动自己的内容。
3. **重读比多读更重要**：做好的笔记需要反复重读，每重读一次都是加深理解、开放新视野的机会。


## 二、完整操作流程（五大步骤）

### 【第一步】选书——建立购书清单

**目的**：有目的地选书，减少与不感兴趣的书的接触，提高选书命中率。

**操作步骤**：

1. **准备“购书清单”**：在笔记本中专门留出一个区域（或用笔记本的前几页），列出想读的书。
2. **收集书讯来源**：
   - 日常生活中随时留意与书相关的信息：报纸书评、朋友推荐、社交媒体推荐、书店偶遇等。
   - 看到感兴趣的书，立刻在笔记本上记下书名、作者、出版社。
   - 可以将剪报、书腰、书单等贴进笔记本。
3. **定期审视清单**：对照清单去书店或网上确认，判断是否真的需要购买。

**产出物**：一份“购书清单”清单（记在笔记本中）


### 【第二步】购书——主动购买

**目的**：带着明确的意图买书，让读书从一开始就充满主动性。

**操作步骤**：

1. 购书前检查自己的购书清单，确认是否仍然需要这本书。
2. 翻阅实体书或试读内容，判断内容是否如笔记记录那般值得购买。
3. 购入后，在笔记本中补全书籍信息（如出版社、购入日期）。

**注意事项**：尽量避免冲动购书，买书前一定对照购书清单。


### 【第三步】读书——三步筛选法

**目的**：从书中优中选优，筛选出真正值得摘抄的内容，为做笔记做好准备。

#### 第一阶段：通读（第一遍筛选）

- 一边阅读，一边把觉得有价值的那一页**折一个角**（或用书签标记）。
- 不要停下来做任何笔记，保持阅读节奏。

#### 第二阶段：重读（第二遍筛选）

- 读完一遍后，回过头来重读折角的那几页。
- 如果仍然觉得很好，就把那一页**另一个角也折起来**（形成双折角）。

#### 第三阶段：标记（第三遍筛选）

- 再次重读所有双折角的页面。
- 如果第三次阅读后仍然觉得值得一读，就**用笔画线做记号**。
- **推荐使用分级画线法**：
  - **普通直线**：表示“重要”“客观上很重要”的内容。
  - **波浪线**：表示“非常重要”“想记住并加以运用”的内容。
  - **圆圈**：圈出关键的专有名词、关键词、关键句。

**产出物**：一本已完成标记的实体书


### 【第四步】记录——制作读书笔记（核心步骤）

**目的**：将书中精华通过“摘抄+感想”的方式内化为自己的精神财富，留下与书交流过的证据。

#### 「葱鲔火锅式」笔记法——标准操作模板

这是奥野宣之最核心的笔记方法。操作步骤如下：

**① 写下笔记基本信息：**
```
[日期]《书名》/ 作者 / 出版社（可选）
```

- 日期格式示例：［180114］（表示2018年1月14日）。

**② 交替进行摘抄与评论：**
- **摘抄**：摘录经过三步筛选后仍然无法割舍的内容。用 **“○”** 标识摘抄部分。
- **评论**：写下自己对这段摘抄的感想、思考或补充说明。用 **“☆”** 标识评论部分。

**笔记范例格式：**

```
［180114］《如何有效阅读一本书》/ 奥野宣之 / 江西人民出版社

○ 摘抄内容：图书都是有重点的，没有必要从头到尾详读。重要的是不管篇幅多少，都要下点功夫让自己对读过的书有点印象。

☆ 我的评论：这段话让我意识到——不必追求读完每一页，哪怕只记住一个观点，也比翻完全书却毫无印象更有价值。以后读书时不再有“必须读完”的心理负担。
```

**三个关键提醒：**
1. **摘抄必须原汁原味**：不要简写或省略，原文的表达本身就蕴含深意。
2. **感想不求完美**：一开始写不出感想，写“好棒！”“原来如此！”也行。重点是开始写。
3. **只摘抄打动你的内容**：即便客观上很重要，如果没有想法、没有共鸣，那就不要摘抄。

**不同类型书籍的处理方法：**
- **纯技巧类/工具书**：可以标记页码直接写下感想，或将重要页面复印替代摘抄。
- **小说/休闲读物**：可以只写整体感想，或标记关键情节页码，写一两句感受即可。
- **内容较少的书**：如果环境允许，可以边读边写笔记，合并第三步和第四步。

**产出物**：一份“书目信息 + 摘抄（○）+ 评论（☆）”的结构化笔记


### 【第五步】活用——输出与重读

**目的**：让笔记不仅成为知识的仓库，更成为灵感和智慧的来源。

#### 具体操作：

**① 定期重读笔记**
- 固定重读频率，建议一周一次或三天一次。
- 每次重读时，可以在原有笔记旁补充新的感想（用不同颜色的笔或标注日期区分）。
- 每重读一次、每增添一条评论，都是加深理解、拓宽新视野的好机会。

**② 参考第三方观点**
- 读完整本书并做好笔记后，去读书点评网站（如豆瓣）阅读他人的书评。
- 换个角度审视自己的读书体验，可以补充在笔记后面。

**③ 系统输出——写一篇完整的书评**
- 将读书过程中产生的零散感想，整合输出为一篇完整的书评。
- 书评可以发表在书籍评论页、自媒体账号或社群中。
- 正式输出的好处：刻意打磨写作技巧、获得他人反馈、形成交流和互动。

**④ 建立笔记检索系统**
- 当笔记本积累到多本后，给笔记本编号。
- 建立一个电子表格，列出每本笔记本上记录的书名、作者、出版社、关键词等，方便日后查找。
- 也可以在每本笔记扉页添加目录，或贴纸胶带做标签。

**产出物**：书评文章、检索用电子表格


## 三、全流程速查清单

| 步骤 | 核心动作 | 产出物 | 关键工具 |
|------|---------|--------|---------|
| ①选书 | 收集书讯，建立购书清单 | 购书清单 | 笔记本 |
| ②购书 | 对照清单，主动购买 | 实体书 | 购书清单 |
| ③读书 | 通读→重读→标记（三步筛选） | 已标记的书 | 折角 + 分级画线 |
| ④记录 | ○摘抄 + ☆评论，制作笔记 | 葱鲔火锅式笔记 | 笔记本 |
| ⑤活用 | 重读笔记→输出书评→建立索引 | 书评 + 检索表 | 笔记本 + 电子表格 |


## 四、常见问题与避坑提醒

| 问题 | 解决方案 |
|------|---------|
| “觉得摘抄太麻烦” | 摘抄本身就是加深记忆的最佳方式。从最简单的做起——哪怕只抄一句话。 |
| “写不出感想” | 从“好棒”“真厉害”开始，关键是养成写的习惯。 |
| “笔记太慢了，读不完” | 并非每本书都值得做全套笔记。工具书可只记要点，休闲书可只写整体感想。 |
| “笔记记完就忘了” | 必须定期重读。建议每周或每三天固定翻阅一次笔记。 |
| “笔记本多了找不到” | 给笔记本编号，用电子表格建立索引。 |


**最后记住一句话：** “读书方法的改变会带来思考方式的转变。”现在，打开一本空白笔记本，写下第一条购书清单吧。
